***************
Getting Started
***************

In this chapter we instruct you how to get started with the extension.

How to...

* Configure a new `API Configuration` record with the basic needed configuration.
* Create and configure a new `API Data Link` content element, select the API endpoint and upload your own template file.
* Create a simple Fluid template to display the API data.
